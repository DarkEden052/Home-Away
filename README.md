# 🏠 PG Finder Web Application :-- HOME AWAY

A responsive and user-friendly web application to discover Paying Guest (PG) accommodations across major cities in India. Built using **HTML**, **CSS**, and **Bootstrap**, this project helps users explore properties, view details, and log in/sign up for a personalized experience.

🚀 **Live Demo:** [Click here to view the live website](https://darkeden052.github.io/Home-Away/)  
📂 **Frontend Only** | 🛠️ **Built with HTML, CSS, Bootstrap** | 🌐 **Hosted on GitHub Pages**

## ✨ Features

- **Responsive Design** using Bootstrap
- **Home Page** with:
  - Clickable Logo in the Left of the Header which redirect to Home Page
  - **Login** and **Signup** buttons on the right
  - Section showcasing **Major Cities**
  - **Footer** with quick links to cities and contact/info
- **Signup & Login** forms to simulate user authentication (front-end only)
- **Dashboard Page**:
  - Visible after login/signup
  - Lists PG accommodations based on city
  - Each PG card includes **location, rent, features, and contact details**
- Clean and intuitive UI focused on usability

## 🛠️ Tech Stack

- **HTML5** – for structure and semantics  
- **CSS3** – for custom styling  
- **Bootstrap 5** – for responsive design and layout
